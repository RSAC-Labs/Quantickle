Node types are organized into "Domains" (not to be confused with Internet domains). These are themed sets of node types (people, places, servers, etc.). You can add your own if you want. Turn on only the sets you need so your menus stay short.

Turn domains on

  • Open View → Type Definitions.
  • Tick the checkboxes for the domains you want. The Default domain stays on to provide basic shapes.
  • Use Reload all if you add new domain files to your workspace and want them to appear. (This can be sluggish)

Autoloading domains

  • When graphs are loaded or imported, the loader will attempt to load node domains present in the data.

Changing node type

  • Select a node and open the Node Editor.
  • The Type dropdown lists options from every active domain—pick one to apply the matching icon, color, and defaults.
  • Switching the type later updates the look without changing your node’s connections.
