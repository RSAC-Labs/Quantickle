Graphs can show remote icons and background images. Quantickle checks before loading anything from the web so you stay in control.

What counts as external

  • Node icon or backgroundImage fields that start with http(s):// or file:.
  • Graph background images that point to remote URLs.

When a prompt appears

  • Choose Allow to keep the images and continue loading.
  • Choose Block to strip the URLs; the graph still loads, just without external pictures.
  • The same choice is offered next time you open the file, so you can change your mind later.

Keep things smooth

  • Prefer workspace-relative image paths or built-in icons to avoid prompts.
  • If an image fails to load, try opening the URL in a browser tab to confirm it is reachable.
