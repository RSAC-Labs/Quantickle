Create a graph node to jump between related graphs. You can link to a saved workspace file or a Neo4j source, then open the linked graph with a double-click.

Create a graph node

  • Select a node and, in the Node Editor, change its Type to graph.
  • Click Select graphâ€¦ to choose a saved graph or a Neo4j dataset when available.
  • Give the node a clear label so it is easy to recognize as a link.

Open the linked graph

  • Double-click the graph node to open the target graph. Quantickle remembers the source and key so you return to the same place each time.
  • If the node has no link yet, the double-click creates a fresh graph you can start editing.

Organize graphs

  • Graph nodes can be used to organize graphs in tree structures, and create dashboards.
