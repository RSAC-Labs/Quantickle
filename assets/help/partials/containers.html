Containers keep related nodes together so you can collapse clutter and move groups as one. Think of them as labeled boxes that travel with their contents.

Making a container and adding items

  • Select the nodes you want to group.
  • In the dropdown menu, select Send to container to wrap them in a new container node.
  • If you first select a container, and then press Send to container in the dropdown menu, the node is sent to the selected container.
  • You can also just use an ALT-drag box to enclose the nodes you want.
  • Name the container to make it easy to find later.

Removing items

  • Select nodes and click Move out of container from the context menu.
  • If you want to free all nodes, just select Remove container from the context menu. The nodes stay.

Tidy the inside

  • Select the container and choose Arrange Nodes to straighten its contents.
  • Applying a layout while a container is selected applies the layout to the container only.
  • Containers stack. You can have containers inside containers, each with their own styling, layout and background
  • Containers can also be collapsed. Doubleclick to collapse, move out of the way and come pack to it later.
    New doubleclick returns the container to original position.
